​📖 简介 (Introduction)
​MarkdownWeaver 是一款专为内容创作者打造的 Android 原生 Markdown 编辑器。
​它致力于解决一个核心痛点：在手机上将 Markdown 文章完美发布到微信公众号。
​大多数移动端编辑器在复制到微信后台时，会出现样式丢失（特别是表格和代码块）。MarkdownWeaver 通过内置的 CSS 内联注入引擎，确保在手机端一键复制出的富文本，粘贴即完美。
​MarkdownWeaver is a native Android Markdown editor designed for content creators. It solves a core pain point: perfectly publishing Markdown articles to WeChat Official Accounts on mobile.
​✨ 核心特性 (Features)
​🎨 Gemini 设计语言: 采用 Google Material Design 3 风格，侧滑抽屉、胶囊切换、悬浮底栏，极其优雅。
​🛠️ 微信完美兼容: 独家 WeChatStyleProvider 引擎，自动将 CSS 转为内联样式，彻底修复微信后台表格、引用样式丢失的 Bug。
​🌈 多主题一键切换: 内置 6 款精美主题，实时预览：
​Next (极简)
​Hexo (经典)
​Google (Material)
​Apple (开发者文档风)
​GitHub (程序员风)
​WeChat (护眼模式)
​⚡ 极致轻量: 纯原生 Java 开发，无后台服务，无广告，秒开。
​🔒 隐私安全: 所有数据仅在本地运行，无需联网，不上传任何服务器。
